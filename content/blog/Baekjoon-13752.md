## 3. Solution 2 (C++ 17, memory: 2020KB, time: 4ms)

```cpp
#include <iostream>
#include <string>
#include <string_view>

int main() {
    int n;
    std::cin >> n;
    while (n--) {
        int k;
        std::cin >> k;
        std::string equals(k, '=');
        std::cout << std::string_view(equals) << '\n';
    }
    return 0;
}
```
