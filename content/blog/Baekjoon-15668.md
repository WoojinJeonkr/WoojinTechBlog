## 2. 풀이 (Python, memory: 83948KB, time: 22092ms)

```python
import sys
input = sys.stdin.readline # 구문을 추가하지 않으면 시간초과
N = int(input())
M = [int(input()) for _ in range(N)]
for i in sorted(M):
    print(i)
```
