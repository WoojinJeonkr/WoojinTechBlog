## 2. 정답 코드 (Python, memory: 35196KB, time: 1996ms)

```python
n = int(input())
x, y = [], []
for i in range(n):
    a, b= map(int, input().split())
    x.append(a)
    y.append(b)

print((max(x) - min(x)) * (max(y) - min(y))) 
```
