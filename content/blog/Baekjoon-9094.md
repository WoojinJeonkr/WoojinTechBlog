## 2. Solution (Python, memory: 31256KB, time: 4376ms)

```python
for _ in range(int(input())):
    n, m = map(int, input().split())
    print(sum((a ** 2 + b ** 2 + m) % (a * b) == 0 for a in range(1, n) for b in range(a + 1, n)))
```
