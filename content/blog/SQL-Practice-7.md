## 4. YEAR, MONTH, DAY 함수 사용법

```sql
SELECT YEAR(GETDATE()) AS [YEAR] -- 년
SELECT MONTH(GETDATE()) AS [MONTH] -- 월
SELECT DAY(GETDATE()) AS [DAY] -- 일
```

여기서 주의해야 할 점은 위의 SQL의 반환값이 문자열이 아닌 정수라는 점이다.
